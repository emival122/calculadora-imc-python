# 📖 Manual do Usuário - Calculadora de IMC

Este guia explica como utilizar a aplicação Calculadora de IMC para obter seu Índice de Massa Corporal de forma simples e rápida.

## 🏁 Iniciando
Ao abrir o programa, você verá uma tela com o título **Calculadora de IMC**, dois campos de preenchimento e um botão de cálculo.

## 🛠️ Funcionalidades

### 1. Informar Peso
* No campo **Peso (Kg)**, digite seu peso corporal.
* Exemplo: `70`

### 2. Informar Altura
* No campo **Altura (m)**, digite sua altura em metros.
* Exemplo: `1.75`
* É possível utilizar ponto ou vírgula como separador decimal.

### 3. Calcular IMC
* Clique no botão **Calcular IMC**.
* O resultado será exibido em um cartão logo abaixo dos campos.

### 4. Visualizar Resultado
O sistema exibirá:
* O valor do IMC com duas casas decimais
* A classificação correspondente (Magreza, Normal, Sobrepeso ou Obesidade)
* Uma cor indicativa para facilitar a interpretação

### 5. Mensagens de Erro
Caso algum valor inválido seja digitado:
* Uma mensagem de erro aparecerá no cartão de resultado
* Corrija os dados e tente novamente

---
*Dica: Utilize valores reais e atualizados para obter um resultado mais preciso.*
